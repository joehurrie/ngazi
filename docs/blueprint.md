# **App Name**: Ngazi Digital

## Core Features:

- Landing Page Replication: Replicate the design and layout of the provided HTML landing page using Next.js components.
- Component-Based Architecture: Implement a component-based architecture for easy content updates and page creation.
- Content Management: Enable the client to create, edit, and manage content through a simple CMS such as Contentful.
- Contact Form: A contact form that users can use to send messages to the design agency.

## Style Guidelines:

- Primary color: Neutral dark gray (#171717) to convey a modern and sophisticated feel.
- Background color: White (#FFFFFF) for a clean and spacious aesthetic, allowing content to stand out.
- Accent color: Light gray (#A3A3A3) to provide subtle contrast and highlight interactive elements without overpowering the design.
- Body and headline font: 'Inter', a sans-serif font known for its readability and modern aesthetic, will be used for both body text and headlines.
- Use minimalist line icons from Lucide to maintain a clean and consistent aesthetic.
- Maintain a maximum width of 1400px for content to ensure readability on larger screens. Use a grid layout with vertical lines for a structured feel.
- Incorporate subtle animations on scroll to improve UX and make the page feel more alive.
